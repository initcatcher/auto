# AutoHotkey 학습 프로젝트

이 프로젝트는 AutoHotkey 스크립트 언어를 학습하기 위한 예제 코드들을 포함하고 있습니다.

## 프로젝트 구조

- `basics/`: 기본적인 AutoHotkey 문법과 기능을 학습하는 예제들
- `hotkeys/`: 단축키 관련 예제들
- `gui/`: GUI 관련 예제들
- `automation/`: 자동화 관련 예제들

1. [tidbit tutorial]("https://www.autohotkey.com/docs/v2/Tutorial.htm")
